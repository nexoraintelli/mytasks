# Controle de Clientes SEO

Protótipo em HTML, CSS e JavaScript para organizar:

- informações principais de cada cliente;
- ASINs já otimizados e pendentes;
- percentual de vendas por ASIN;
- rounds de otimização;
- links das planilhas de sugestão;
- etapas completas do fluxo;
- status e progresso de cada cliente.

## Como abrir

1. Extraia o arquivo ZIP.
2. Abra a pasta `controle-seo-clientes`.
3. Clique duas vezes no arquivo `index.html`.

Os dados são salvos no `localStorage` do navegador. Isso significa que:

- funcionam sem banco de dados;
- ficam salvos no mesmo navegador e computador;
- podem ser apagados se os dados do navegador forem limpos.
